    # Layout 布局

    协助进行页面级整体布局。

    # 组件概述：

    • <er-layout> : 布局容器，其下可以嵌套一下的所有组件，包括它自身。
    • <er-header> ：顶部布局，需要自己添加高度，可以嵌套任何元素，只能放在 <er-layout>中
    • <er-sider> ：侧边栏，需要自己添加高度和宽度，可以嵌套任何元素，只能放在 <er-layout>中
    • <er-content> ：内容布局，需要自己添加高度，可以嵌套任何元素，只能放在 <er-layout>中
    • <er-footer> ：底部布局，需要自己添加高度，可以嵌套任何元素，只能放在 <er-layout>中 
    <er-layout>
  <er-header>Header</er-header>
  <er-content>Content</er-content>
  <er-footer>Footer</er-footer>
</er-layout>

    # 引入方式

    <script>
import { Layout, Header, Footer, Content, Sider } from 'vue-erui'
export default {
  components: {
    'er-layout': Layout,
    'er-header': Header,
    'er-content': Content,
    'er-sider': Sider,
    'er-footer': Footer
  }
}
</script>
