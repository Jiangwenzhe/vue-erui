    # Grid 栅格

    24格栅格系统。

    基础栅格： 使用单一的一组 er-row 和 er-col 栅格组件，就可以创建一个基本的栅格系统，所有列（er-col）必须放在 er-row 内。 

    <er-row>
  <er-col span="12">col-12</er-col>
  <er-col span="12">col-12</er-col>
</er-row>

<er-row>
  <er-col span="8">col-8</er-col>
  <er-col span="8">col-8</er-col>
  <er-col span="8">col-8</er-col>
</er-row>

<er-row>
  <er-col span="6">col-6</er-col>
  <er-col span="6">col-6</er-col>
  <er-col span="6">col-6</er-col>
  <er-col span="6">col-6</er-col>
</er-row>

    区块间隔： 栅格常常需要和间隔进行配合，你可以使用 er-row 的 gap 属性。

    左右偏移： 使用 offset 可以将列向右侧偏。例如，offset="3" 将元素向右侧偏移了 3 个列（column）的宽度。 

    <er-row gap="20">
  <er-col span="12">col-12</er-col>
  <er-col span="12">col-12</er-col>
</er-row>

<er-row gap="20">
  <er-col span="8">col-8</er-col>
  <er-col span="8">col-8</er-col>
  <er-col span="8">col-8</er-col>
</er-row>

<er-row gap="20" >
  <er-col span="6">col-6</er-col>
  <er-col span="6">col-6</er-col>
  <er-col span="6">col-6</er-col>
  <er-col span="6">col-6</er-col>
</er-row>

<er-row gap="20"  class="row">
  <er-col span="12" offset="3">col-12 offset-3</er-col>
  <er-col span="6" offset="6">col-6 offset-6</er-col>
</er-row>

    响应式布局： 参照 Bootstrap 的 响应式设计，预设五个响应尺寸：phone pad spc pc bpc (spc是-small-pc-的缩写，表示窄屏电脑，bpc是-big-pc-的缩写，表示大屏电脑)

    TIP

    改变界面宽度试试看把 🤪

    <er-row class="row">
  <er-col span="24" :pc="{span: 12}" :spc="{span: 6}" :pad="{span: 18}">col</er-col>
  <er-col span="24" :pc="{span: 12}" :spc="{span: 18}" :pad="{span: 6}">col</er-col>
</er-row>

<er-row class="row">
  <er-col span="24" :pc="{span: 8}" :spc="{span: 3}" :pad="{span: 10}">col</er-col>
  <er-col span="24" :pc="{span: 8}" :spc="{span: 18}" :pad="{span: 4}">col</er-col>
  <er-col span="24" :pc="{span: 8}" :spc="{span: 3}" :pad="{span: 10}">col</er-col>
</er-row>

<er-row class="row">
  <er-col span="24" :pc="{span: 6}" :spc="{span: 2}" :pad="{span: 10}">col</er-col>
  <er-col span="24" :pc="{span: 6}" :spc="{span: 2}" :pad="{span: 10}">col</er-col>
  <er-col span="24" :pc="{span: 6}" :spc="{span: 10}" :pad="{span: 2}">col</er-col>
  <er-col span="24" :pc="{span: 6}" :spc="{span: 10}" :pad="{span: 2}">col</er-col>
</er-row>

    # 引入方式

    <script>

export default {
  components: {
    'er-row': Row,
    'er-col' : Col
  }
}
</script>
