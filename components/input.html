    # 输入框

    输入框的各种形态： 默认输入框，只读输入框，禁用输入框，error输入框 

    <er-input v-model="msg"></er-input>
<p>{{ msg }}</p>

<er-input value="readonly" readonly></er-input>
<er-input value="disabled" disabled></er-input>
<er-input value="wrong input" error="您的输入错了"></er-input>

    # 引入方式

    <script>
import { Input } from 'vue-erui'
export default {
  components: {
    'er-input': Input
  },
  data() {
    return {
      msg: '今天的风儿甚是喧嚣呢'
    }
  }
}
</script>
